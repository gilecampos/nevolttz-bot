# Nevolttz Bot - Landing Page 1.0
Esse projeto é uma Landing page de um bot de discord que ainda está em desenvolvimento, projeto voltado para desmonstrar a utilidade do bot e por que você deve utiliza-lo
em seu servidor.

# Aprendizado
Desenvolvendo essa Landing page eu aprendi muito sobre efeito, e pseudo-elementos em CSS além de expandir meus conhecimentos em mobileFirst e mediaqueries.

# Tecnologias Usadas:

- JavaScript
- Git e Github
